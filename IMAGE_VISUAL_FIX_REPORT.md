## 🎨 設計上の重要な決定事項

### 画像解決の優先順位

```
1. wikimedia-images.json を優先的に参照（getOnsenImage()）
   ↓
2. 見つからない場合のみ images.ts のフォールバックを使用
   ↓
3. それでも見つからない場合は Unsplash Source API でキーワード検索
```

**結論:**
- ✅ `images.ts` のプレースホルダー状態は、wikimediaが充実していれば問題ない
- ✅ wikimedia-images.json を充実させることが最優先

---

### JSON First ポリシーの維持

すべてのテキスト・リンク・画像参照は `content.json` / `texts.json` で管理:

- ✅ コンポーネント内にハードコードなし
- ✅ 画像キーは文字列またはオブジェクト形式（`{ "key": "...", "keywords": "..." }`）
- ✅ 型安全性を維持（TypeScript の ImageReference 型）
