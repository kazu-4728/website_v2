# 🎨 ベンチマークサイト風デザイン大改造計画

**作成日**: 2025年12月21日  
**目的**: 「あえの風」レベルのベンチマークサイトに寄せるための大改造

---

## 🚨 現状の問題点

### 1. ビジュアルインパクトの不足
- 背景画像が適切に表示されていない（Wikimedia画像の制約）
- 画像サイズが小さく、視覚的インパクトが弱い
- カラーパレットが暗すぎる（dark-950/dark-900中心）

### 2. 画像の問題
- 収集された画像が少ない（5箇所のみ）
- 実際の温泉風景の高品質画像が不足
- Hero背景画像が正しく設定されていない

### 3. レイアウトの問題
- セクション間の余白が不十分
- 画像と文字のバランスが悪い
- モバイル表示の最適化が不完全

---

## 🎯 ベンチマークサイトの特徴分析

### 「あえの風」レベルのサイトが持つ要素

1. **圧倒的なビジュアル**
   - 高品質な写真（2400px以上）
   - 全画面を使ったHero（100vh+）
   - パララックス効果とシネマティック演出

2. **和のテイスト**
   - 落ち着いた配色（ベージュ・紺・白）
   - 和紙・水引・伝統模様のモチーフ
   - 温かみのあるトーン

3. **情報密度とナビゲーション**
   - 予約・空室状況への明確な導線
   - プラン一覧の分かりやすい表示
   - 多言語対応

4. **体験価値のアピール**
   - 料理・施設・自然の魅力を写真で伝える
   - 「行ってみたい」気持ちを醸成
   - ストーリー性のあるコンテンツ

## 📋 大改造実施計画

### Phase 1: 画像システムの完全刷新（最優先）🔴

#### 1.1 高品質画像の収集
- [ ] Unsplash API を使用した温泉画像の収集
  - 検索キーワード: "onsen", "hot spring", "Japanese bath", "ryokan"
  - 必要枚数: 各温泉地につき3-5枚（最低30枚）
- [ ] Pexels API を使用した補完画像
- [ ] Wikimedia Commons から高解像度画像の再取得

#### 1.2 画像データの整理
- [ ] `data/onsen-images.json` 作成
  - 各温泉地ごとにカテゴリ分け
  - Hero用、カード用、詳細ページ用に分類
- [ ] 画像のメタデータ管理
  - ライセンス情報
  - クレジット表記
  - 推奨サイズと用途

#### 1.3 テーマ設定への反映
- [ ] `themes/onsen-kanto/content.json` の画像URLを更新
  - Hero背景画像の変更（高品質・高解像度）
  - カード画像の差し替え（実際の温泉風景）
  - CTA背景の追加

---

### Phase 2: カラーパレットの刷新🟡

#### 2.1 和のテイストに合わせた配色
```css
// 現在（暗すぎる）
dark-950: #0a0a0f
dark-900: #1a1a23
primary-500: #f97316 (オレンジ)

// 提案（温かみのある和風）
warm-beige: #f5f1e8 (背景ベース)
deep-indigo: #2d3748 (アクセント暗部)
gold-accent: #d4af37 (金色アクセント)
onsen-blue: #4a90a4 (温泉の青)
warm-orange: #e87722 (温かみのあるオレンジ)
```

#### 2.2 背景色の変更
- [ ] `bg-dark-950` → `bg-warm-beige` または `bg-gradient-to-b from-warm-beige to-white`
- [ ] セクションごとに色を変える（単調さを避ける）
- [ ] オーバーレイのグラデーション調整（暗すぎない）

---

### Phase 3: レイアウトの大改造🟢

#### 3.1 Hero セクション
- [ ] 背景画像を2-3枚のスライドショーに変更
- [ ] オーバーレイの明るさ調整（from-black/70 → from-black/40）
- [ ] タイトルの文字色を白から金色に変更（目立たせる）
- [ ] CTA ボタンを大きく（min-w-[280px]）

#### 3.2 カードコンポーネント
- [ ] AreaSelection: カードの角丸を増やす（rounded-3xl）
- [ ] GridGallery: 画像の明るさ調整（オーバーレイ薄く）
- [ ] RecommendedOnsen: ホバー時の影を強化

#### 3.3 セクション間の余白
- [ ] py-32 → py-40 md:py-48（より大きな余白）
- [ ] セクション間に装飾線を追加（和風のボーダー）

---

### Phase 4: タイポグラフィの改善🔵

#### 4.1 フォント
- [ ] Google Fonts から和風フォント追加
  - Noto Serif JP（見出し用）
  - Zen Maru Gothic（本文用）
- [ ] `app/layout.tsx` でフォント設定

#### 4.2 見出しサイズ
- [ ] Hero: text-9xl → text-[10rem]（さらに大きく）
- [ ] Section: text-7xl → text-8xl
- [ ] 英語サブタイトル: 文字間隔を広げる（tracking-widest）

---

### Phase 5: アニメーションの強化⚪

#### 5.1 スクロール連動
- [ ] IntersectionObserver を使用した段階的表示
- [ ] 各セクションがビューに入ったときにフェードイン
- [ ] 画像のパララックス効果（背景が遅く動く）

#### 5.2 ホバーエフェクト
- [ ] カードのホバー時に光が走る効果
- [ ] ボタンホバー時のリップルエフェクト
