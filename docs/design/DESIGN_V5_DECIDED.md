# 設計構想 V5 - 決定版

作成日: 2025年12月10日  
ベンチマーク: 和倉温泉 あえの風 公式サイト（デザイン例として参考）  
目標: 画像メインの情報多彩な温泉紹介サイト

## 📋 決定事項

### ページ構成

**現在実装するページ**:
- `/` - ホーム（Hero + エリア選択 + おすすめ + 検索・絞り込み + 温泉地一覧）
- `/[slug]` - 温泉地詳細

**将来的に追加するページ**（今は実装しない）:
- `/plans` - おすすめプラン一覧
- `/plans/[slug]` - プラン詳細
- `/articles` - 特集記事一覧
- `/articles/[slug]` - 記事詳細
- `/contact` - お問い合わせ

### ホームページの構成

1. **Hero** - フルスクリーン、複数画像スライダー、大胆なオーバーレイ
2. **エリアから探す** - 大画像カード、ホバーエフェクト強化
3. **おすすめ温泉** - 大画像ギャラリー、複数画像表示
4. **検索・絞り込み機能** - エリア、泉質、効能、日帰り/宿泊、アクセス方法
5. **温泉地一覧** - 大画像カード、ホバーエフェクト

### 検索・絞り込み機能

**実装する機能**:
- エリア検索
- 泉質フィルタ
- 効能フィルタ
- 日帰り/宿泊フィルタ
- アクセス方法フィルタ

### 温泉詳細ページの構成

1. **Hero** - フルスクリーン、複数画像スライダー
2. **概要・魅力** - 大画像 + テキスト、視覚的リズム
3. **温泉データ** - アイコン付きカード、視覚的に分かりやすく
4. **画像ギャラリー** - 複数画像のスライダー、大胆な配置
5. **宿泊施設一覧** - 複数施設、大画像カード、予約ボタン
6. **周辺観光スポット** - 大画像カード、アクセス情報
7. **地図** - インタラクティブ、周辺情報表示
8. **季節ごとの魅力** - 季節ごとの画像、視覚的に表現
9. **予約サイトへのリンク** - 明確なCTAボタン
10. **前後ナビゲーション**
