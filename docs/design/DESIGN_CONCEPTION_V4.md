# 設計構想 V4 - ベンチマークサイトへの最適化計画

作成日: 2025年1月  
ベンチマーク: 和倉温泉 あえの風 公式サイト（https://www.aenokaze.jp/）

## 1. 設計構想の概要

### 1.1 目標

**「あえの風」レベルの世界観・クオリティを実現しつつ、多拠点温泉ガイドとして実用性を高める**

- **世界観**: 上質感・洗練されたデザイン
- **実用性**: 本当に使える温泉ガイド（予約導線、詳細情報、比較機能）
- **JSON First**: すべてのコンテンツとUIテキストをJSONで管理

### 1.2 ベンチマークサイトの核心要素

1. **視覚的インパクト**
   - 高品質な実写画像（温泉、客室、食事）
   - スライダーで複数画像を表示
   - 画像にオーバーレイテキストで情報を追加

2. **情報設計**
   - 明確な予約導線（各ページに予約ボタン）
   - 詳細なプラン情報（料金、日程、含まれるもの）
   - アクセス情報（地図、交通手段、駐車場）

3. **ユーザー体験**
   - スムーズなスクロールアニメーション
   - ホバーエフェクト（画像ズーム、カードリフト）
   - 直感的なナビゲーション

## 2. フェーズ別実装計画

**実装内容**:

**完了条件**:
- すべての温泉地に適切な画像が表示される
- 画像クレジットが正しく表示される

### フェーズ3: 温泉詳細ページの充実（優先度: 高）

**目的**: 本当に使える温泉ガイドとして必要な情報を提供
