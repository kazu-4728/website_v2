# 無料画像API一覧

作成日: 2025年1月

## 無料で使える画像API

### 1. Unsplash API
- **URL**: https://unsplash.com/developers
- **無料プラン**: 50リクエスト/時間
- **登録**: 必要（無料）
- **特徴**: 高品質な写真、商用利用可能
- **検索**: 可能
- **日本語対応**: 限定的

### 2. Pexels API
- **URL**: https://www.pexels.com/api/
- **無料プラン**: 200リクエスト/時間
- **登録**: 必要（無料）
- **特徴**: 高品質な写真・動画、商用利用可能
- **検索**: 可能
- **日本語対応**: 限定的

### 3. Pixabay API
- **URL**: https://pixabay.com/api/docs/
- **無料プラン**: 制限なし（レート制限あり）
- **登録**: 必要（無料）
- **特徴**: 写真・イラスト・ベクター、商用利用可能
- **検索**: 可能
- **日本語対応**: あり

### 4. Wikimedia Commons API
- **URL**: https://www.mediawiki.org/wiki/API:Main_page
- **無料プラン**: 制限なし
- **登録**: 不要
- **特徴**: 教育・商用利用可能、多言語対応
- **検索**: 可能
- **日本語対応**: あり（既に使用中）

### 5. Flickr API
- **URL**: https://www.flickr.com/services/api/
- **無料プラン**: 3,600リクエスト/日
- **登録**: 必要（無料）
- **特徴**: ユーザー投稿写真、ライセンス多様
- **検索**: 可能（ライセンスフィルタ可能）
- **日本語対応**: あり

### 6. Openverse (旧CC Search)
- **URL**: https://api.openverse.engineering/
- **無料プラン**: 制限なし
- **登録**: 不要（APIキー推奨）
- **特徴**: Creative Commonsライセンスの画像検索
- **検索**: 可能
- **日本語対応**: 限定的

## 推奨される統合順序

1. **Wikimedia Commons API** (既に使用中)
   - 最も確実、日本語対応良好

2. **Pixabay API**
   - 無料プランが充実、日本語対応あり
   - レート制限が緩い

3. **Pexels API**
   - 高品質、無料プランが充実

4. **Unsplash API**
   - 高品質だが、レート制限が厳しい

5. **Flickr API**
   - 多様なライセンス、日本語対応あり

## 実装方針

1. 複数のAPIを順番に試す（フォールバック）
2. 各APIで「rotenburo」「露天風呂」「bath」などのキーワードで検索
3. 最も適切な画像を選択
4. ライセンス情報を保存
