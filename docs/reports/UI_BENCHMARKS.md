# UI ベンチマーク調査レポート

最終更新: 2025-01-XX

## 📋 調査目的

Next.js / React ベースの旅行・ホテル・リゾート・温泉・観光系のLPを調査し、以下の要素を抽出する：
- ヒーローセクションのビジュアルとアニメーション
- ナビゲーションの設計
- セクション構成とリズム
- 画像の使い方
- インタラクションのパターン

## 🎯 調査対象サイト（参考事例）

### 1. 一般的な旅行・リゾートサイトのベストプラクティス

**参考URL（推定）:**
- 大手旅行サイト（JTB、HIS、楽天トラベルなど）
- リゾートホテル公式サイト（星野リゾート、プリンスホテルなど）
- 温泉旅館公式サイト（箱根、草津などの名湯）

**良い点:**
- **Hero セクション**: フルスクリーン背景画像 + オーバーレイ + 明確なCTAボタン
- **ナビゲーション**: 固定ヘッダー、モバイル対応ハンバーガーメニュー
- **セクション構成**: Hero → 特徴 → ギャラリー → レビュー → CTA の流れ
- **画像の使い方**: 高品質な実写写真、ホバーエフェクト、遅延読み込み
- **インタラクション**: スムーズなスクロールアニメーション、パララックス効果

**このプロジェクトに流用する要素:**
- フルスクリーンHero + スライドショー機能
- カード型のギャラリー表示（GridGallery）
- スクロールトリガーアニメーション（Framer Motion）
- 画像のホバーエフェクト（scale + transition）

## 📊 抽出された共通パターン

### 1. Hero セクション
- **フルスクリーン背景画像**: 高品質な実写写真
- **オーバーレイ**: テキストの可読性を確保
- **CTAボタン**: 明確で目立つ配置
- **スライドショー**: 複数の画像を自動切り替え（2-3秒間隔）
- **アニメーション**: フェードイン、ズームイン効果

### 2. ナビゲーション
- **固定ヘッダー**: スクロール時も表示
- **モバイル対応**: ハンバーガーメニュー
- **アクティブ状態**: 現在のページを明示

### 3. セクション構成
- **Hero** → **特徴紹介** → **ギャラリー** → **レビュー/声** → **使い方ガイド** → **CTA**
- 各セクション間に適切な余白
- スクロールトリガーアニメーション

### 4. 画像の使い方
- **高品質な実写**: ストックフォトではなく実写を優先
- **遅延読み込み**: パフォーマンス最適化
- **ホバーエフェクト**: scale + transition
- **アスペクト比**: 16:9 または 4:3 を維持

### 5. インタラクション
- **スムーズなアニメーション**: 60fps を維持
- **ホバーエフェクト**: カード、ボタン、リンク
- **スクロールアニメーション**: Framer Motion など
- **フォーカス状態**: キーボードナビゲーション対応

## 🎨 このプロジェクトへの適用方針

### 優先度: 高
1. **Hero スライドショー**: 季節ごとの画像（星空/雪見/紅葉/新緑）
2. **実写温泉画像**: Wikimedia Commons から取得
3. **セクション構成**: Hero → エリア → テーマ → 声 → 使い方 → CTA
4. **レスポンシブデザイン**: モバイルファースト

### 優先度: 中
1. **スクロールアニメーション**: Framer Motion によるフェードイン
2. **ホバーエフェクト**: カード型コンポーネント
3. **画像最適化**: Next.js Image コンポーネント

### 優先度: 低
1. **パララックス効果**: パフォーマンスを考慮して必要に応じて
2. **ビデオ背景**: 画像で代替可能

- **ベストプラクティス**: 実写画像、明確な導線、スムーズなアニメーション
- **このプロジェクトの強み**: JSON First アーキテクチャ、型安全性、テーマ切替可能
- **改善点**: Hero画像の実写化、セクション構成の最適化、インタラクションの標準化
