### 1. [AGENT_GUIDE.md](./AGENT_GUIDE.md) ⭐ **最優先**
エージェント向けの包括的なガイドです。作業を開始する前に**必ず読んでください**。

**主な内容**:
- 画像取得と使用のワークフロー

- ファイルの編集権限
- 作業フロー

### 2. [START_HERE.md](./START_HERE.md)

### 3. [../FUTURE_TASKS.md](../FUTURE_TASKS.md)

## 🔄 作業フロー

1. **作業開始前**
   - `AGENT_GUIDE.md`を読む
   - `START_HERE.md`を読む
   - `FUTURE_TASKS.md`で課題を確認

2. **設計の確認**
   - `../design/DESIGN_CONCEPTION_V4.md`で設計構想を確認
   - JSON Firstアーキテクチャを理解

3. **実装**
   - 小さな変更から始める
   - JSON Firstアーキテクチャを維持
   - 型安全性を確保

4. **画像取得の場合**
   - 画像取得スクリプトを実行
   - 取得結果をユーザーに報告
   - **ユーザーの承認を待つ**
   - 承認後にサイトに反映

## ⚠️ 重要な原則

### 画像の取得と使用
- **画像を自動的にサイトに反映しない**
- 取得した画像は必ずユーザーに報告し、確認を得る
- ユーザーの承認を得てから、サイトに反映

### JSON Firstアーキテクチャ
- ハードコードされたテキストを追加しない
- すべてのテキストは`themes/onsen-kanto/texts.json`で管理
- コンテンツ構造は`themes/onsen-kanto/content.json`で管理
