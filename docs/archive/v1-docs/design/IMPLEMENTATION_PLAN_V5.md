## 📋 実装方針

### 基本方針

1. **既存サイトを忘れる**: 大改造のため、既存の実装は参考程度に
2. **ベンチマークサイトを目指す**: 「あえの風」レベルの世界観・クオリティ
3. **画像メイン**: 高品質な実写画像を大胆に使用
4. **JSON First**: すべてのコンテンツとUIテキストをJSONで管理（維持）
5. **テンプレート化**: テーマ切替可能な設計を維持

## 📝 実装の詳細

### 新規コンポーネント

1. **`FullscreenHero`** (`app/components/home/FullscreenHero.tsx`)
   - フルスクリーンHero（複数画像スライダー）
   - オーバーレイテキスト
   - パララックス効果

2. **`ImageGallery`** (`app/components/ui/ImageGallery.tsx`)
   - 画像ギャラリー（複数画像のスライダー）
   - サムネイル表示
   - ライトボックス表示

3. **`LargeImageCard`** (`app/components/ui/LargeImageCard.tsx`)
   - 大画像カード（ホバーエフェクト強化）
   - オーバーレイテキスト

4. **`AccommodationCard`** (`app/components/onsen/AccommodationCard.tsx`)
   - 宿泊施設カード（大画像、予約ボタン）

5. **`NearbySpotCard`** (`app/components/onsen/NearbySpotCard.tsx`)
   - 周辺観光スポットカード（大画像、アクセス情報）

6. **`BookingButton`** (`app/components/ui/BookingButton.tsx`)
   - 予約ボタン（明確なCTA）

7. **`SearchBar`** (`app/components/search/SearchBar.tsx`)
   - 検索バー

8. **`FilterPanel`** (`app/components/search/FilterPanel.tsx`)
   - フィルタパネル

9. **`ParallaxSection`** (`app/components/ui/ParallaxSection.tsx`)
   - パララックス効果セクション

### 既存コンポーネントの強化

1. **`CinematicHero`** → **`FullscreenHero`** に強化
2. **`GridGallery`** → **`LargeImageGallery`** に強化
3. **`Testimonials`** → アバター画像、背景画像、星評価を追加
