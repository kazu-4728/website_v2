## 📝 使用方法

### content.jsonでの画像指定

```json
{
  "bgImage": { "key": "main", "keywords": "onsen,hot spring,japan" }
}
```

または

```json
{
  "image": "hakone"  // 簡易形式
}
```

### 画像の追加方法

1. `app/lib/images.ts`の`ONSEN_KANTO_IMAGES`に画像URLを追加
2. `content.json`でキーを指定
